Posted on 2018-08-31 |

1. 鸡鸭相邻

农场有 n 只鸡鸭排位一个队伍，鸡用”C表示，鸭用”D”表示。当鸡鸭挨着时会产生矛盾，需要对所排的队伍进行调整，使鸡鸭各在一边，每次调整只能让相邻的鸡和鸭交换位置，现在需要尽快完成队伍调整，你需要计算出最少需要调整多少次可以让上述情况最少。例如 CCDCC->CCCDC->CCCCD这样就能使之前的两处鸡鸭相邻变为一处鸡鸭相邻，需要调整队形两次。

输入描述：
输入一个长度为N，且只包含C和D的非空字符串

输出描述：
使得最后仅有一处鸡鸭相邻，最少的交换次数

输入例子：
CCDCC
DDCC

输出例子：
2
4

由于D只能和C交换，所以这个排序是稳定的
假定所有的D都移动到最后一个位置，然后把最后位置的D都依次向前移动0-(n-1)次就是要排序的结果，n是D的个数

早睡带来的舒适感

Posted on 2018-08-22 |

8月19号收假，从老家回到西安的学校已经是晚上11点20分了，赶紧洗漱一番，冲了个澡，跟另一半互道晚安之后就立刻睡觉了，12点前入睡的，没有刷手机，没有乱思绪，就那样累的睡着了。第二天醒来的时候感觉特别的舒适，空气与皮肤接触的感觉特别的好，就好像自己置身于仙境一般。如此，20号，21号晚都是在12点入睡了，每天早上7点50起床，基本8个小时的睡眠，中午稍微睡上10多分钟，一天都是充满精力的，早上照镜子，脸上的大痘痘也基本消下去了。

如果每天都这样，在12点前睡觉，持续一个月，那么我就是坚持了一个月；如果持续一年，那么我就坚持了一年，那么我就养成了早睡的习惯，早睡就成了我身体中的一部分。没有了手机的诱惑，没有了思绪乱飞的失眠，有了长足的睡眠，这对于我们这些年轻人是非常珍贵的一件事。而我从这连续三天的体验中，感受到切实的幸福感，充沛了精力，如果我们每天花一部分时间去做自己觉得有意义的事情，那么我们就会成为有意义的事情。你每天相同的状态，就是你多年的状态。

任何事情，要么现在做，要么今天做。 原地不动，大篇长论，没有实践，我们就不知道我们的方向对错与否，不要拖延，错了就及时修正，对了就继续前行。

初识动态规划

Posted on 2017-12-25 |

动态规划：
通常是基于一个或多个初始状态，当前子问题的解由上一个子问题的解推出

其结构由 状态 和 转移方程 构成

最少硬币
如果我们有面值为1元，3元和5元若干枚，如何用最少的硬币凑够11元？

用f(i) = j 来表示凑够 i 元需要 j 个硬币
f(0) = 0
f(1) = f(1 - 1) + 1 = f(0) + 1 = 1
f(2) = f(2 - 1) + 1 = f(1) + 1 = 2
对于f(3)有两种方案
f(3) = f(3 - 1) + 1 = f(2) + 1 = 3
f(3) = f(3 - 3) + 1 = f(0) + 1 = 1
那么状态转移方程为
f(3) = min{f(3 - 1) + 1, f(3 - 3) + 1}
即
f(i) = min{d(i - Vj) + 1}, 其中 i - Vj > 0, Vj 表示第 j 个硬币的面值

输入 install package
搜索 package resource viewer
