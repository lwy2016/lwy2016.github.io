Posted on 2017-07-14 |

1. 简介

JavaScript是一种专为与网页交互而设计的脚本语言，由下列三种不同部分组成：
ECMAScript 提供核心语言功能
文档对象模型(DOM) 提供访问和操作网页内容的方法和接口
浏览器对象模型(BOM) 提供与浏览器交互的方法和接口

3. 基本概念

5种原始数据类型： Undefined, Null, Boolean, Number, String
1种复杂数据类型： Object,

把非数值转换为数值： Number(any type), ParseInt(string), parseFloat(string)
