树的一些通用解法：
跟树相关的题，90%的都可以用递归解决，一般需要一个helper辅助函数来进行递归，递归都会有递归终止条件
相对于迭代解决问题，递归代码量较少且清晰简便

// 先序操作
helper(root.left);
// 中序操作
helper(root.right);
// 后续操作
