sublime下Java开发

Posted on 2020-03-10 |

1.文件命名为 runJava.bat 且保存在 jdk/bin 目录下

2.修改 sublime 的运行方式

2.1 安装 Package Resource Viewer

ctrl+shift+p 打开 package control
输入 install package
搜索 package resource viewer

2.2 修改 Javac.sublime-build 文件

ctrl+shift+p 打开 package control
输入 open resource
回车后 输入 java
回车后 选择 Javac.sublime-build 就能够打开我们需要修改的文件

3.Ctrl+B 运行即可
