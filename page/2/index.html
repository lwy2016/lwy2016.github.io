初识动态规划

Posted on 2017-12-25 |

动态规划：
通常是基于一个或多个初始状态，当前子问题的解由上一个子问题的解推出

其结构由 状态 和 转移方程 构成

最少硬币
如果我们有面值为1元，3元和5元若干枚，如何用最少的硬币凑够11元？

用f(i) = j 来表示凑够 i 元需要 j 个硬币
f(0) = 0
f(1) = f(1 - 1) + 1 = f(0) + 1 = 1
f(2) = f(2 - 1) + 1 = f(1) + 1 = 2
对于f(3)有两种方案
f(3) = f(3 - 1) + 1 = f(2) + 1 = 3
f(3) = f(3 - 3) + 1 = f(0) + 1 = 1
那么状态转移方程为
f(3) = min{f(3 - 1) + 1, f(3 - 3) + 1}
即
f(i) = min{d(i - Vj) + 1}, 其中 i - Vj > 0, Vj 表示第 j 个硬币的面值

1.文件命名为 runJava.bat 且保存在 jdk/bin 目录下

ctrl+shift+p 打开 package control
输入 install package

  • 人生有两个悲剧：一是万念俱灰，一是踌躇满志。

  • 中国有句老话，叫做取法于中，仅得其下，工作上要向高标准看齐，生活上则要向低标准看齐。
  • 意大利经济学家帕累托归纳出一个统计结论，这个地球上百分之二十的人口享有百分之八十的财富。
  • 约旦国王侯赛因之所以能得享天年，就是因为他谨慎，没人说得清这世界上想杀他的人有多少，而张子强就是因为老大当惯了，所以很快就因为一命呜呼了。他入狱后手下的人还想去劫狱劫法场，在大陆这根本不可能！

人生首飞

Posted on 2017-09-24 |

虽然说出差太多时候是累的，但我内心还是希望自己能够获得每次去出差的机会，多见见外面的世界、涨涨视野。19日下午5点我们一行来到吉源国际乘坐机场大巴，去往西安-咸阳国际机场。说来也怪，我只晕大巴车，尤其是在睡眠不足及饮食质量差的情况下，匆匆吃了片晕车药，等待到达机场。第一次坐飞机，虽谈不上激动，但让我回忆起了2005年我在乡里读初一的时候，看到从乡里到县城的公交车，希冀自己将来什么时候能够坐上公交车，第二年去了县城上学，每两周回家一次，做了6年，对我来说已是寻常事。恰如当时考上大学去郑州玩几天回家，特意做一次火车了解坐火车的流程一样，现在火车票已经积攒了厚厚的一匝。又如大一跟着舍友做地铁，特意的留意该怎么怎么走，而现在早已是轻车熟路。动车高铁虽傻傻分不清楚，但也做过了许多次，确实节省了许多时间，也希望自己毕业3年内的工资能够像动车高铁的每小时所需的费用一样高，这是我的一个目标呀。飞机的速度很快，从西安到榆林，路途不算远，1个半小时左右，飞机通过长长的滑道，逐渐升入高空，在云层上空飞行没多久，还没完全感受到飞行的快乐，它就开始渐渐的降落了，不过也不影响它是一个 prefect flight experience.
